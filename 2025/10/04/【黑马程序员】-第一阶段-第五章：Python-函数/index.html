4.2 函数返回值之 None 类型

思考：如果函数没有使用 return 语句返回数据，那么函数有返回值吗？

  • 实际上是：有的。

Python 中有一个特殊的字面量 None，其类型是 <class 'NoneType'>。
无返回值的函数，实际上就是返回了 None 这个字面量。

None 表示空的、无实际意义的意思。函数返回的是 None，就表示这个函数没有返回什么有意义的内容，也就是返回了 空 的意思。

七、变量在函数中的作用域

变量作用域指的是变量的作用范围（变量在哪里可用，在哪里不可用）。

主要分为两类：局部变量和全局变量。
